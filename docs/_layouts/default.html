GitCrisp

A clean, focused Git GUI

Visual commit graph, per-hunk staging, full management dialogs for branches, remotes, submodules, and tags — built with Python and Qt.

Download View on GitHub

Features

📊

Visual Commit Graph

Lane-based graph showing branching and merging history with lazy pagination and branch navigation.

🧩

Per-Hunk Staging

Stage or unstage individual diff hunks within a file. Discard hunks or whole files inline.

🌿

Branch Management

Sidebar tree, graph context menu, and a dedicated dialog for upstream tracking, rename, and the checkout-conflict reset prompt.

🏷️

Tag Management

Create lightweight or annotated tags from any commit, delete tags, and push individual tags to a remote.

📦

Stash Management

One-click stash with confirmation, view stash contents inline, pop, apply, or drop from context menu.

🌐

Remote Operations

Push, pull, fetch, and a Remotes CRUD dialog. All network ops run in background threads with status logging.

🔗

Submodule Support

Manage submodules from a dedicated dialog, recursive clone by default, and click submodule diffs to jump into the submodule repo.

📂

Multi-Repository

Switch between repositories with persistent open and recent lists. Clone new repos from URL.

🎨

Theming & Insights

Light and dark themes with a custom typography scale, plus per-author commit stats over a configurable date range.
